# BioFilter

A 2D top-down tower defense game built with Godot 4 (C#).

Defend a bunker's air filtration system from Bio Particles. Place filter modules and shape the airflow path — but be careful, every wall you add restricts airflow. Too many choke points and your people suffocate.
